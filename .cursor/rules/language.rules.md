# Language Rules – Project Conventions

## Idiomas

- **Código**  
  - Inglés  
  - Incluye: nombres de variables, funciones, clases, interfaces, enums, archivos `.ts`, `.tsx`, `.js`

- **Comentarios inline en código**
  - Inglés

- **Documentación**
  - Español  
  - Incluye: archivos `.md`, README, documentación técnica, explicaciones funcionales

- **UI / UX**
  - Español  
  - Incluye: textos visibles al usuario, labels, placeholders, mensajes de error, títulos, botones
